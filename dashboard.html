How do I set this up?

One-time Firebase setup (free, ~3 minutes)

  1. Go to console.firebase.google.com and create a new project (name it anything, e.g. "speakup-tracker")
  2. In the left sidebar, click Build > Realtime Database
  3. Click Create Database, choose any region, then select Start in test mode
  4. Copy the database URL shown at the top (looks like https://speakup-tracker-xxxxx-default-rtdb.firebaseio.com)
  5. Paste that URL in the box above and click Connect
  6. In index.html, find the line window._fbDb='https://YOUR-PROJECT-ID...' near the bottom and replace it with: window._fbDb='YOUR-URL/speakup'
  7. That's it! The app on their phones will start sending data and you'll see it here

Note: "test mode" rules expire after 30 days. When they do, go to Database > Rules and set read/write to true.
